60 FPS cheats master list for PSP games.
Last updated on: April 03, 2017.

Special thanks to user GamerzHell9137 for maintaining the original FPS master list at the PPSSPP forum.

Anything regarding this list and its cheats should be posted in the discussion thread at the PPSSPP forum.

How to search for codes:
Press CTRL+F and type the name of the game in the search field of your web browser. If you can't find the game listed here or in the discussion thread then there may not be one for it at the moment.

How to use CW cheats with PPSSPP:
1: Close PPSSPP, locate the PPSSPP folder on your system.
2: Go to the folder: PPSSPP > MEMSTICK > PSP.
3: If it doesn't exists, create a new folder here named "Cheats" (without quotes).
4: Open PPSSPP and goto Settings > System, then "Enable Cheats" should be turned ON.
5: Open the game you wish to add cheats to.
6: Close PPSSPP and go into the Cheats folder you had created. There should be an INI file in there now with the name of your game ID (ULUS123/ULES456...).
7: Open the INI file (Notepad or any text editor can do this).
8: Depending on your game, paste into your INI file the lines of codes found after clicking your game link from the list below.
9: Open game in PPSSPP again, play it, pause, then go into the PPSSPP cheats menu. If the newly added cheat doesn't show up:
a- Make sure the INI file saved and isn't blank.
b- Check and see if you put the right Cheat code into the right INI folder.

Important Notes:
- The "BETA" label next to a game means that the cheat/hack works OK but has minor/moderate issues that affect the gameplay; in other words, it's not perfect. Click on the game post and it should contain details about its issues.
- You must set the PSP CPU speed to at least 333Mhz (PPSSPP/PSP/PSVita) to guarantee framerate stability with most of these cheats. Some games even need higher speeds (PPSSPP only, of course) and if that's the case with a game, the cheat details will specify it.
- If are going to report an issue for a game to PPSSPP developers, please make sure that a FPS cheat is not the cause of it.
